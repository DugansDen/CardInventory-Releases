# Card Inventory

A desktop application for managing and tracking your card collection, with cert lookups (PSA/CGC) and eBay price checking.

## Download

Download the latest version from the [Releases](https://github.com/DuganD22/CardInventory-Releases/releases) page.

## Installation

1. Download the latest `.zip` file from Releases
2. Extract the zip to a folder of your choice
3. Run `CardInventory.exe`

## Features

- Track your card inventory with detailed information
- PSA and CGC certificate lookup
- eBay price lookups for market value
- Export and manage your collection
